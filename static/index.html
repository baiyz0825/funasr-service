服务配置说明

模型下载目录
FunASR AutoModel 不支持 cache_dir 参数，通过环境变量控制下载路径：
MODELSCOPE_CACHE - ModelScope 模型
HF_HOME - HuggingFace 模型

默认下载到 ~/.cache/modelscope/。如需自定义目录，启动前设置：
export MODELSCOPE_CACHE=/your/path
CPU 多核推理
FunASR 默认仅使用 4 线程，本服务自动检测 CPU 物理核心数并配置（上限16线程）。

通过 ncpu 参数可手动控制：
POST /v1/config 设置 "ncpu": 8

建议值：min(物理核心数, 16)
多模型同时加载
支持同时加载多个模型（默认最多2个），不同场景使用不同模型无需切换。

资源不足时会提示卸载。可通过配置调整上限：
POST /v1/config 设置 "max_loaded": 3
启动自动加载
在「模型管理」页面设置自动加载的模型，服务重启后自动加载。

配置持久化在 config.json，也可通过 API 管理：
GET/POST /v1/config
